1.我觉得哪个概念很难或者至关重要。哪些概念，是天才的概念。  
2.用自己的话和代码去解释为什么这个概念是天才的？  
-这些概念解决了什么问题？  
-如果没有这些概念，我们会被卡在哪一步？  
3.结合数学史，设身处地地设想，自己是18世纪的数学家，面对这个问题，我能想到什么？
